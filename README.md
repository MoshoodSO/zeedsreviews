## ğŸš€ Getting Started

1. **Clone the repository**

   ```sh
   git clone https://github.com/MoshoodSO/zeedsreviews.git
   ````

2. **Navigate into directory**

   ```sh
   cd zeedsreviews
   ```

3. **Install dependencies**

   ```sh
   npm install
   ```

   or

   ```sh
   yarn install
   ```

4. **Create environment variables**

   Copy `.env.example` (if present) to `.env` and populate with your Supabase credentials and other configuration.

5. **Start development server**

   ```sh
   npm run dev
   ```

   or

   ```sh
   yarn dev
   ```

6. Open your browser at `http://localhost:3000` to see the app in action.
